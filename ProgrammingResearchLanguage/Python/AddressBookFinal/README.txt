Final Project

Purpose: Demonstrate use of python accessing a database, writing to a file, use of OO and languages data structures.

Requirements:
- Python 2.7 x86 (including pyunit)
- MySQL 5.6 server of greater
- All scripts must be in a single directory
- Database schema and user matching those described in database_final.py

Installation/Execution:

-Open Administrator CMD window in directory (Alt+F, M, A) or open Terminal for Mac/Linux
-Type: python executor_final.py
